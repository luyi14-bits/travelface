前提条件

  • 项目已上传到 GitHub 仓库：luyi14-bits/travelface

本地部署

cd travelface
pip install -r requirements.txt
python -m streamlit run app.py

Streamlit Cloud 部署

1 登录：浏览器访问 share.streamlit.io，使用 GitHub 账号登录
2 新建应用：点击右上角 "New app" 按钮
3 配置部署信息：
配置项值
Repositoryluyi14-bits/travelface
Branchmain
Main file pathapp.py
Python Version3.12（推荐）
4 设置 Secrets（可选）：点击 "Advanced settings"，在 Secrets 中以 TOML 格式添加： 
OPENAI_API_KEY = "sk-你的API密钥"
LLM_BASE_URL = "https://ark.cn-beijing.volces.com/api/v3"
LLM_MODEL_NAME = "doubao-1.5-pro-32k"
⚠ 注意：必须使用 TOML 格式（KEY = "value"），不能用 .env 格式（KEY=value）。不配置 API Key 也可以正常使用，系统会自动切换到内置模拟推荐数据。
5 部署：点击 "Deploy" 按钮，等待 1-2 分钟即可完成。

更新部署

代码推送到 GitHub 后，在 Streamlit Cloud 管理界面点击 "Reboot" 即可拉取最新代码并重新部署。

故障排除

依赖安装失败

  • 确认 requirements.txt 中无 opencv-python-headless、mediapipe、deepface、tensorflow 等编译型依赖
  • 若之前部署过旧版本，建议点击 "Clear build cache" 清除缓存后再 Reboot

运行时错误

  • 确认 Secrets 格式为 TOML（KEY = "value"），不是 .env 格式（KEY=value）
  • 查看 Streamlit Cloud 日志获取具体错误信息

人脸检测未框出人脸

  • 确保照片中肤色区域足够大且光照正常
  • 肤色检测在极端光照或非人脸肤色物体上可能失败，系统会自动降级
