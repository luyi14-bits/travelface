# TravelFace Streamlit Cloud 部署指南

## 前提条件

1. 项目已上传到 GitHub 仓库：https://github.com/luyi14-bits/travelface
2. 拥有 GitHub 账号

## 本地部署

```bash
cd travelface
pip install -r requirements.txt
python -m streamlit run app.py

# Windows 用户也可以双击 run.bat 一键启动
```

**依赖清单（全部为纯 Python 包，零编译）**：

| 包名 | 用途 |
|------|------|
| `streamlit` | Web 框架 |
| `numpy` | 矩阵运算 |
| `Pillow` | 图像处理 |
| `openai` | LLM API |
| `requests` | HTTP 客户端 |
| `python-dotenv` | 环境变量管理 |

## Streamlit Cloud 部署

### 1. 登录
浏览器访问 https://share.streamlit.io/ ，使用 GitHub 账号登录

### 2. 新建应用
点击右上角 **"New app"** 按钮

### 3. 配置部署信息

| 配置项 | 值 |
|--------|-----|
| **Repository** | `luyi14-bits/travelface` |
| **Branch** | `main` |
| **Main file path** | `app.py` |
| **Python Version** | 3.12（推荐） |

### 4. 设置 Secrets（可选）

点击 **"Advanced settings"**，在 **Secrets** 中以 TOML 格式添加：

```toml
OPENAI_API_KEY = "sk-你的API密钥"
LLM_BASE_URL = "https://ark.cn-beijing.volces.com/api/v3"
LLM_MODEL_NAME = "doubao-1.5-pro-32k"
```

> **注意**：必须使用 TOML 格式（`KEY = "value"`），不能用 `.env` 格式（`KEY=value`）。不配置 API Key 也可以正常使用，系统会自动切换到内置模拟推荐数据。

### 5. 部署
点击 **"Deploy"** 按钮，等待 1-2 分钟即可完成。

## 部署优势

| 特性 | 说明 |
|------|------|
| **零模型下载** | 全部视觉分析基于纯 NumPy，无需下载任何预训练权重 |
| **轻量依赖** | 仅 6 个纯 Python 包，总安装体积 < 50MB |
| **无编译** | 零 C 扩展，任意 Python 版本均可安装 |
| **首次部署快** | 无需下载模型文件，比 DeepFace 方案快 10 倍以上 |

## 资源限制

Streamlit Cloud 免费版限制：
- 内存：1GB
- 存储：1GB
- 部署时间：30 天（需定期重新部署）
- 并发用户数：有限

本项目完全在免费版限制内运行。

## 配置说明

| 环境变量 | 说明 | 是否必需 |
|---------|------|----------|
| `OPENAI_API_KEY` | OpenAI 兼容 API 密钥 | 推荐（无则使用模拟数据） |
| `LLM_BASE_URL` | API Base URL | 仅非 OpenAI 官方 API 时需要 |
| `LLM_MODEL_NAME` | 模型名称 | 可选，默认豆包 doubao-1.5-pro-32k |

## 更新部署

代码推送到 GitHub 后，在 Streamlit Cloud 管理界面点击 **"Reboot"** 即可拉取最新代码并重新部署。

## 故障排除

### 依赖安装失败
- 确认 `requirements.txt` 中无 `opencv-python-headless`、`mediapipe`、`deepface`、`tensorflow` 等编译型依赖
- 若之前部署过旧版本，建议点击 **"Clear build cache"** 清除缓存后再 Reboot

### 运行时错误
- 确认 Secrets 格式为 TOML（`KEY = "value"`），不是 `.env` 格式（`KEY=value`）
- 查看 Streamlit Cloud 日志具体错误信息

### 人脸检测未框出人脸
- 确保照片中肤色区域足够大且光照正常
- 肤色检测在极端光照或非人脸肤色物体上可能失败，此时系统会降级为中央区域猜测
