🔍 人脸检测

NumPy YCbCr 肤色 Flood Fill

纯 NumPy 实现，将 RGB 转换到 YCbCr 色彩空间提取肤色掩膜， 再用 Flood Fill 找连通域，经六层校验筛选最优人脸区域。

👤 用户画像 & 情绪

NumPy 亮度 RGB 通道比 对比度

基于人脸裁剪区域的亮度梯度估年龄、RGB 红润度推性别、 亮度+对比度判情绪，全部基于 NumPy 图像特征。

✈️ 旅行推荐

OpenAI API Mock Fallback

基于用户画像和情绪生成个性化旅行路线， API 不可用时自动降级到 12 条内置模拟路线。

🎨 UI 展示

Streamlit Light/Dark 主题 Hover 动效

暗色/浅色双主题自动适配，按钮缩放 1.04x、 Metric 上移 2px、旅行卡片上移 4px 等悬停动效。
